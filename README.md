## Getting Started


### Prerequisites

- Docker
- Docker Compose

### Installation

1. Clone the repository:
```sh
git clone https://github.com/yourusername/short-and-sweet.git
```
2. Navigate to the project 'src' directory
```sh
cd short_and_sweet/src
```
3. Run installation command
```sh
docker-compose --build
```

### How to run

If you have a sufficient computing power (16 GB of ram and ok processor exp. AMD Ryzen 5 5600X) run this simple command

```sh
docker-compose up
```
If however you don't currently hold such a computing power, worry not this application has been designed to be run on the cloud. Head to `short_and_sweet/src/ai` library and checkout how to deploy machine learning endpoints on the cloud
